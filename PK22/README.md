# PK2.2 联机扩展自定义包

## 实现功能
- 过回合自动存档

## 下载方式
1. 直接将 ***"联机扩展.cpp"*** 自定义包文件移动到`pk2.2/script/002 其他自定义包`路径下。
2. 可以用 **PKHH修改器** 的自定义包功能检查是否成功开启
   - 注意：1.13.0及以上版本的PKHH中需要把联机扩展自定义包加入到设定集中并应用
3. 启动游戏不报错即可。

## 开启联机扩展后的联机流程
（需要同时开启联机工具的自动上传+自动下载）

### 联机准备工作
1. 一名玩家在游戏中创建新剧本，选择多个势力并进入游戏。
2. 所有玩家右键成员列表中自己的名字，修改自己的君主名。
3. 从初始玩家开始进行游戏：

### 游戏进行
1. 开始：轮到玩家回合时，会自动下载存档并弹窗提示回合开始。
2. 载入：***打开右键菜单，点击快速读档***。
3. 进行：进行回合内操作。
4. 结束：***游戏内过回合***， 会自动在下一个玩家回合开始前存档并上传。
