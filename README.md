- 理论上支持所有 **三国志11** 的 MOD

## 下载方式

- ### Release版本（推荐）

1. 安装.NET 8.0 桌面运行时：https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0
2. 推荐直接下载Release下的最新版本压缩包，解压后无需安装，直接运行。
3. 服务端包含windows/linux两个版本，请跟据平台下载对应的压缩包。

- ### 运行时自包含版

1. 如果没有安装.NET 8.0 运行时，可以直接下载Self-Contain版本（体积较大），解压后直接运行。

- ### 源码编译

1. 下载源码：`git clone https://github.com/pdt012/San11PVPTool3.git`
2. 自行编译。
3. [代码贡献方式](#代码贡献指南)。

## 使用方式

### 1. 准备局域网环境

可以使用云服务器或花生壳等内网穿透工具搭建局域网环境。

### 2. 配置服务端

打开 `San11PVPToolServer/appsettings.json` 文件

修改 `Urls` 配置，例如：
```
"Urls": "http://192.168.1.100:31177"
```

运行服务端程序。

### 3. 配置客户端

1. 启动`San11PVPToolClient`客户端程序。
2. 点击 **配置连接** ，设置用户名与服务端地址，服务端地址格式为`ip:port`，例如`127.0.0.1:31177`。
   - 如果地址正确且连接成功，点击 **刷新** 时就不会弹窗报错
3. 点击 **设置** ，选择存档所在的SaveData文件夹，不同MOD可能略有不同。
   - 原版三国志11在`我的文档/Koei/San11( Tc)/SaveData`
   - pk2.2在`pk2.2/Koei/San11/SaveData`
   - 血色在`San11Pk/Koei/San11 Tc/SaveData`

### 4. 创建/加入房间

### 5. 联机准备工作
1. 一名玩家在游戏中创建新剧本，选择多个势力并进入游戏
2. 保存存档至 31号存档（必须使用此编号）
3. 点击 **上传** 将存档上传至服务端

### 6. 游戏进行
1. 下载：轮到玩家回合时，点击 **下载** 获取最新存档。
2. 载入：在游戏中载入31号存档。
3. 进行：进行回合内操作。
4. 存档：游戏内过回合， ***当下一个玩家回合开始时*** ，保存存档到31号。
   - 工具会跟据当前游戏内君主名判断属于哪位玩家的回合，
   - 如果在自己的回合内上传了存档，可能导致自动上传/下载失效。
5. 上传：点击 **上传** 。
6. 通知下一个玩家开始回合，重复以上步骤。

### * 自动上传/下载
1. 点击 **设置** ，勾选 **自动上传** / **自动下载**。
2. 存档到31号后，等一两秒会自动上传（如果成功会有信息提示）。
3. 右键房间成员列表，点击 **修改君主名** 选择游戏中自己的君主名（繁中版请输入繁体名）。
4. 当服务端存档的当前君主与选择的君主相同时会自动下载存档。
